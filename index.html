Codenames: Duet

Cooperative multiplayer online guessing game

An online multiplayer cooperative game based on the board game "Codenames: Duet". Initially built using HTML5, CSS, Bootstrap 4, JavaScript, OOP and jQuery for the front-end display and Google Firebase to handle the online database.

Re-built using React to better program players and game board functionalities. A player lobby, player-cap system and card randomization were added to the project.

Vinyl Sales

Online vinyl record shopping catalog

An online vinyl record sales catalog with full CRUD (Create, Read, Update, Delete) functionality. Created utilizing HTML5, CSS, Bootstrap 4, OOP, JavaScript and React on the front-end display. A MySQL database and PHP scripts were used to provide back-end functionality.

Initially started as a read & add project, I later added in update and delete functionality as well as React Routing in order to expand the scope of the shopping catalog.

Photo Pilgrim

Local trending photo discovery & sharing social media platform

A personal idea which evolved into a 3-week final team project. Photo Pilgrim allows users to search for the top trending photos based on inputted location. Created utilizing HTML5, CSS, Bootstrap 4, OOP, JavaScript and React on the front-end display. A MySQL database and PHP scripts were used to provide back-end functionality.

For this project I was elected to be both Team Lead and Front End Lead. I designed and wrote code for the major Home, Search and Photo Details pages. After the original 3 week deadline was over, I continued to add to the project by implementing React Routing, multi-platform responsiveness and Amazon A3 server for photo storage.

About me
